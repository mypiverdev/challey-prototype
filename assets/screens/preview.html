Architecture
Complete Flow Map
How every screen connects.
App Store → Download → 01 Splash → 02 Join or Create

PATH A (Kid):
02 → 03a Enter Codes → 03b Set PIN → 03c Confirm PIN → Kid Dashboard
Returning: 03d Enter PIN → Dashboard
Forgot: 03d → 03e Forgot PIN → Parent notified

PATH B (Parent Onboarding):
02 → 04a Auth Method → 04b Email Form (or social skip) → 04c Pick Role → 04d Name Family → 04e Choose Pet → 04f Welcome

PATH C (Parent Initial Setup):
04f → 05a Empty Dashboard → 05b Add Kids → 05c Kids Success → 05d Pick Chores → 05e Pick Rewards → 05f First Post → 05g Parent Dashboard (Live)

PARENT DASHBOARD (5 Tabs):
05g Dashboard ↔ 06a Verify ↔ 06b Rewards ↔ 06e Living Room ↔ 06f Family
Settings: 06g Settings → 06h Companion Config
Notifications: Bell icon → 06m Notifications Panel

PARENT ACTIONS (Sub-Screens):
Rewards tab → 06c Create Reward | 06k Wishlists
Dashboard → 06d Create Chore | 06i Swap Review
Family tab → 06j Shout-out | 06l Give Badge
